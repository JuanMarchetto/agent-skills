# Agent Skills Marketplace

Plugin marketplace for production-grade AI coding assistant skills.

## Install

Add the marketplace (one time):
```
/plugin marketplace add JuanMarchetto/agent-skills-marketplace
```

Then install any skill:
```
/plugin install maestro-mobile-testing@agent-skills-marketplace
/plugin install demo-pipeline@agent-skills-marketplace
/plugin install agent-browser@agent-skills-marketplace
/plugin install webreel@agent-skills-marketplace
```

## Available Skills

| Skill | Description | Repo |
|-------|-------------|------|
| **maestro-mobile-testing** | Maestro E2E testing for React Native/Expo — 11 patterns, auth, CI/CD, MCP | [repo](https://github.com/JuanMarchetto/maestro-mobile-testing) |
| **demo-pipeline** | Natural language to automated demo recording for mobile and web | [repo](https://github.com/JuanMarchetto/demo-pipeline) |
| **agent-browser** | Browser automation via inference.sh — navigate, scrape, record | [repo](https://github.com/JuanMarchetto/agent-browser-skill) |
| **webreel** | Scripted browser demo videos with cursor animation and overlays | [repo](https://github.com/JuanMarchetto/webreel-skill) |
