# Agent Skills Marketplace

Plugin marketplace for [agent-skills](https://github.com/JuanMarchetto/agent-skills).

## Install

```bash
/plugin marketplace add JuanMarchetto/agent-skills-marketplace
/plugin install agent-skills@agent-skills-marketplace
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| **agent-skills** | Mobile E2E testing, demo recording pipelines, browser automation, and scripted video capture |
