# iOS Glass UI Designer

Redesign mobile app UI to feel unmistakably iOS-native with modern Apple-like glass materials (translucency, blur, depth). Glass is used as a tool for hierarchy and focus, not as decoration.

## Install

```
/plugin marketplace add JuanMarchetto/agent-skills
/plugin install ios-glass-ui-designer@agent-skills
```

Or via [skills.sh](https://skills.sh):
```bash
npx skills add JuanMarchetto/ios-glass-ui-skill
```

Or manually:
```bash
git clone https://github.com/JuanMarchetto/ios-glass-ui-skill.git
cp -r ios-glass-ui-skill ~/.claude/skills/ios-glass-ui-designer
```

## What's Inside

- Design philosophy: native over custom, restraint over spectacle
- iOS glass material system (ultra-thin, regular, thick)
- Visual style guide (typography, color, depth, lighting)
- Layout and structure patterns (safe-area aware, touch targets)
- Component principles (buttons, lists, navigation, modals/sheets)
- Interaction and motion guidelines
- Output requirements for each redesigned screen
- Decision-making rules and avoid list
