# YouTube DevRel

Create developer YouTube content, technical screencasts, and video tutorials. Covers tutorial structure, screen recording setup, live coding, shorts vs long-form, thumbnails, and SEO for technical content.

## Install

```
/plugin marketplace add JuanMarchetto/agent-skills
/plugin install youtube-devrel@agent-skills
```

Or via [skills.sh](https://skills.sh):
```bash
npx skills add JuanMarchetto/youtube-devrel-skill
```

Or manually:
```bash
git clone https://github.com/JuanMarchetto/youtube-devrel-skill.git
cp -r youtube-devrel-skill ~/.claude/skills/youtube-devrel
```

## What's Inside

- Developer audience breakdown (learners, workers, seniors, decision makers)
- Tutorial template with hook, pacing, and chapter structure
- Screen recording setup (equipment, settings, workflow)
- Live coding formats and best practices
- Shorts vs long-form strategy
- Thumbnail design principles for technical content
- YouTube SEO (titles, descriptions, keywords)
- Content calendar templates
- Getting started checklist
